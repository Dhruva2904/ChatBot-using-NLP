# ChatBot-using-NLP
Implementation of ChatBot using NLP (AICTE Project)
# Overview
A simple chatbot using NLP with nltk, scikit-learn, and streamlit.
# Installation
pip install nltk scikit-learn streamlit
# Usage
streamlit run chatbot.py
# Dependencies
nltk
scikit-learn
streamlit

import nltk
